# ABB Technikerschule - DTC Kurs

Dieses Repository enthält die Projektdateien und Übungen für das Modul DTC (Development Toolchain) an der ABB Technikerschule in Baden.

## Thema Git

Ein einfache Kotlin-Projekt zum Üben von Git-Grundlagen und Versionskontrolle:
- **RollTheDice**: Ein Würfelspiel-Simulator

Dieses Projekt dient dazu, Git-Workflows, Branching, Merging und Collaboration zu erlernen.

## Voraussetzungen

- Kotlin installiert
- Git
- IntelliJ IDEA oder eine andere Java-IDE (optional)
